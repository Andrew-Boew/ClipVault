# ClipVault

Менеджер истории буфера обмена для macOS. Работает в фоне через иконку в менюбаре: сохраняет всё скопированное и вставляет обратно по глобальному хоткею. macOS хранит только последнее скопированное — ClipVault хранит историю.

## Возможности

**История копирований**
- Автоматически сохраняет текст, ссылки, картинки и файлы
- Определяет приложение-источник каждой записи
- Дедупликация: повторное копирование того же содержимого не создаёт дубль (SHA-256)

**Вставка по хоткею**
- <kbd>⌘ ⇧ V</kbd> открывает плавающую панель истории поверх любого приложения
- Навигация стрелками, <kbd>Enter</kbd> — вставка в активное приложение
- Двойной клик — вставка выбранной записи
- Мультивыбор через <kbd>⌘</kbd>-клик: несколько файлов вставляются вместе, тексты склеиваются

**Поиск**
- Фильтрация истории по содержимому и приложению-источнику по мере ввода

**Закрепление**
- Правый клик → Pin: запись всегда отображается вверху списка
- Закреплённые записи не удаляются лимитами и кнопкой Clear History

**Скриншоты**
- Новый скриншот (<kbd>⌘ ⇧ 4</kbd>) автоматически копируется в буфер — сразу готов к вставке

**Лимиты истории**
- Максимальное количество записей (по умолчанию 500)
- Максимальный возраст записей (по умолчанию 7 дней)
- Очистка выполняется при запуске и после каждого сохранения

**Настройки**
- Смена хоткея
- Лимиты истории
- Список исключённых приложений
- Автозапуск при входе в систему (SMAppService)

## Приватность

- Все данные хранятся локально в `~/Library/Application Support`: база SwiftData и PNG-файлы картинок. Сетевых запросов нет
- Исключённые приложения: содержимое, скопированное из них, не записывается в историю
- Clear History удаляет записи из базы и файлы картинок с диска
- Ограничение: текст, скопированный вручную через ⌘C, неотличим от любого другого текста — определить в нём пароль невозможно. Для приложений с чувствительными данными используется список исключений

## Требования

- macOS 14+
- Разрешение Accessibility — используется только для эмуляции <kbd>⌘V</kbd> при вставке

## Сборка

```bash
git clone <адрес репозитория>
open ClipVault.xcodeproj
```

Сборка и запуск: ⌘R. Зависимость [KeyboardShortcuts](https://github.com/sindresorhus/KeyboardShortcuts) устанавливается автоматически через Swift Package Manager.

При первой вставке macOS запросит разрешение: Системные настройки → Конфиденциальность и безопасность → Универсальный доступ → включить ClipVault.

## Стек
